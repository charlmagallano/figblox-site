Documentation

Export from Figma. Import in Roblox Studio.

Getting started

Install the Figma community plugin and the Roblox Studio plugin. Open your UI frame in Figma, run Export, then paste or load the bundle in Studio to preview changes before confirming.

Prefixes

Use layer name prefixes to control Roblox classes (for example .textbutton for a TextButton). If you omit a prefix, Figblox creates a Frame by default.

Smart Merge

After you edit the design in Figma, export again. New and changed layers update in Studio; other Studio-only edits on unchanged layers are preserved.

Images

Raster and image fills export as assets. Match asset names in Roblox so Figblox can link decals automatically.
