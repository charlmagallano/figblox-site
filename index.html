Features

✓

Smart re-importing

Changed layout values can update without wiping untouched Studio edits. Export again after design changes. Figblox merges instead of replacing everything.

✓

Layout that translates

Figma auto layout becomes UIListLayout, UIGridLayout, and UIPadding. Spacing, alignment, padding, and direction carry over automatically.

✓

Scaling & device layout

To Scale, Lock Text, and Lock Stroke handle responsive behavior. Confirming an import runs the usual tool chain. You do not need to run those manually every time.

✓

Export, images & preview

JSON and bundled images from Figma; paste into Studio and open Preview first. [+] new, [~] changed, [=] in sync. Then Confirm.
