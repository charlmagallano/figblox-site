Platform overview

More than a Figma exporter.

Figblox is a two-plugin system that moves your UI from Figma into Roblox Studio — no rebuilding by hand. Merge updates without throwing away the scripts and tweaks you added in Studio.

  • Prefix system for exact Roblox class control
  • Import preview — see every change before it lands
  • Image deduplication and auto-linking
  • History with undo/redo inside the plugin
  • One-click components: Button, ScrollView, Toggle

Prefix Reference

Put prefixes at the start of a layer name to control what Roblox object Figblox creates.

Object Prefixes

.frameFrame
.textlabelTextLabel
.textbuttonTextButton
.textboxTextBox
.imagelabelImageLabel
.imagebuttonImageButton
.scrollingframe / .scrollvVertical ScrollingFrame
.scrollingframex / .scrollhHorizontal ScrollingFrame
.viewportframeViewportFrame
.canvas / .canvasgroupCanvasGroup

Modifier Prefixes

.rasterRenders the layer as a PNG
.gray / .grayscaleExports as grayscale PNG
.ignoreSkips layer during export
.lockApplies lock behavior
.parentMakes this child the Roblox parent of its group
.hover / .clicked / .pressed / .toggled / .disabledState markers
.textlabel Health Bar
.imagebutton Play Button
.gray .scrollh .raster Card

Built-in Tools

To Scale

Converts px values to percentages for responsive UI across screen sizes.

Lock Text

Keeps text size fixed when the UI scales.

Lock Stroke

Keeps UIStroke thickness from scaling.

Link Images

Matches exported images to Roblox Decals by name and links them.

You don’t need to run these manually. Confirming an import handles them automatically.

One-click Components

Make Button

Turns a Frame or ImageLabel into a clickable button.

Make TextBox

Converts a TextLabel into an editable text field.

Make ScrollView

Wraps a Frame into a ScrollingFrame with scrollable content.

Make Toggle

Creates a toggle from a Frame with On/Off child frames.
