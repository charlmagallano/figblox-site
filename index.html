Features

Export in One Click

Click Export in the Figma plugin. Figblox packages your entire design as JSON and image assets instantly.

Smart Merge

Re-export after updating your design. Figblox merges changes — new layers appear, changed layers update, and your Studio edits stay untouched.

Auto Layout → UIListLayout

Figma auto layout maps directly to UIListLayout, UIGridLayout, and UIPadding. Spacing, direction, and alignment carry over automatically.

Responsive by Default

To Scale converts pixel values to percentages. Lock Text and Lock Stroke keep UI crisp at every resolution. All applied automatically on Confirm.

Platform overview

More than an exporter.

Figblox is a full round-trip workflow. Design, export, preview every change before it lands, confirm, and re-import whenever you update.

  • Import preview — [+] new [~] changed [=] in sync

  • Undo/redo history inside the plugin
  • One-click components: Button, ScrollView, Toggle
